Readme Document for the session_summary_writer.rb and test_result_html.rb

To use these libraries the user must install amrita. To install amrita go to the amrita-1.0.2 directory and run the command "ruby install.rb".

These libraries allows the user to create the session summary and test result html files. An example of how to use each library read is provided in html_file_writer_tester.rb. For additional information and available functions check the source code in session_summary_writer.rb, test_result_html, and html_writer.rb.
